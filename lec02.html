2. Explain cache organization in multi-core CPUs (shared vs separate caches) (3 marks)
Answer
  • Shared coherent cache: Cores share a single cache (e.g., L2 or L3).
    • Advantage: Better data sharing between cores and reduced data duplication.
  • Separate caches: Each core has its own private cache.
    • Requirement: Needs a coherence protocol to maintain consistency.
    • Advantage: Reduces contention for cache resources.
  • In all cases, cores share the interconnect to the rest of the system.

1. Describe the four stages of evolution (4 marks)
Answer
  1. Single-threading: One task processes at a time. No hardware parallelism.
  2. Multitasking/Multithreading: Multiple tasks appear to run at once via context switching (time-slicing) on a single CPU.
  3. Hyper-Threading (HT): Two logical threads execute simultaneously on one physical core by sharing execution units.
  4. Multi-core Technology: Multiple physical cores on one chip, dividing work effectively for true parallel execution.
